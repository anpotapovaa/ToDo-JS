# ToDo App (JavaScript)

Учебный проект — простой ToDo-лист, выполненный в процессе изучения JavaScript, HTML и CSS.  
Проект направлен на практику работы с DOM, обработку событий и работу с локальным хранилищем.

## Технологии

- HTML  
- CSS  
- JavaScript (ES6+)

## Функциональность

- создание и ввод новых задач  
- удаление задач  
- очистка всех задач  
- фильтрация задач (выполненные, невыполненные, все)  
- таймер в секундах для выполнения задачи  
- сохранение данных в LocalStorage (локальное хранение)

## О проекте

Проект выполнен в рамках учебного курса.

Основная логика ToDo приложения прорабатывалась с поддержкой ИИ-ассистента, а верстка и стилизация — самостоятельно

Цель проекта — практика работы с JavaScript, управление DOM, обработка пользовательских событий и использование локального хранилища для сохранения данных.

## Возможные улучшения

- улучшить дизайн и стилизацию  
- добавить адаптивность  
- расширить функционал (например, категории, приоритеты, drag & drop)  
- вынести логику в отдельные модули для лучшей читаемости
